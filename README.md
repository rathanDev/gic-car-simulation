### GIC Auto Driving Car Simulation


# Technologies
1. Java 17
2. Spring Boot 3.4.4
3. Junit
4. Maven
5. Logback


# Installation
1. Clone repository:
> git clone https://github.com/rathanDev/gic-car-simulation.git

2. Navigate to directory
> cd gic-car-simulation

3. Build project:
> mvn clean install

4. Run test
> mvn test

5. Run application:
> java -jar .\target\gic-car-simulation-0.0.1-SNAPSHOT.jar

6. Check logs
> cat .\logs\app.log
